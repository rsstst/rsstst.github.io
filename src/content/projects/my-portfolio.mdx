Disini saya sempat bingung, kira - kira framework apa yang cocok untuk website portfolio. Faktor yang saya pertimbangkan itu kecepatan loading, simpel, dan bisa diatur kontennya dengan mudah. Jadilah AstroJS digunakan untuk membuat website ini.

<br />

AstroJS saya pilih karena dia merupakan framework ringan yang menghasilkan website statis namun memiliki package yang cukup banyak. Saya menemukan framework ini dari JAMSTACK dan setelah membaca tentang Static Site Generation (SSG), saya merasa kalau framework ini cocok untuk use-case saya ini. Saya memanfaatkan fitur content collection dan support Astro terhadap MDX dalam mengatur konten di website ini. Untuk styling, saya menggunakan TailwindCSS karena saya familiar dengan CSS ini dan memanfaatkan sedikit SCSS di beberapa bagian, terutama di Breadcrumbs.
