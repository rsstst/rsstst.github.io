Projek ini dibuat saat saya mengikuti mata kuliah Augmented Reality. Projek ini dibuat dengan memanfaatkan HTML, CSS, dan AR.js. AR.js adalah sebuah library JavaScript yang digunakan untuk membuat aplikasi augmented reality berbasis web. Library ini memungkinkan pengembang untuk membuat pengalaman AR yang dapat diakses melalui browser web tanpa perlu menginstal aplikasi tambahan. 

<br />

Model yang ada saat ini berupa model 3D sederhana tanpa warna gedung baru Institut Teknologi dan Bisnis ASIA Malang yang dibuat menggunakan AutoCAD. Model ditampilkan saat kamera diarahkan ke marker yang telah ditentukan. Model tersebut dapat di=zoom dan di-rotate baik pada mobile maupun desktop.
