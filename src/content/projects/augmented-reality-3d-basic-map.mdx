In this project, I created an Augmented Reality (AR) application that renders a 3D model utilizing HTML and AR.js.
The application allows users to display and interact with a 3D model by pointing their device's camera at a specific marker.
The 3D model itself is a simple and basic representation of Institut Teknologi dan Bisnis ASIA Malang layout, created using AutoCAD and exported as a .glb file.
