In this project, I developed a news portal website named Jendela Tekno Dunia.
The tech used in this project is NextJS framework for the frontend and Strapi CMS for the backend to manage the content.
The website features a clean and minimalistic design, allowing users to easily navigate and read the latest news articles related to technology.
The content is dynamically fetched from the Strapi CMS, enabling easy updates and management of news articles.
The website is responsive and optimized for both desktop and mobile devices.

Currently, the project is running with Strapi free tier, so it may take a while to load the content.
