Website ini dibuat saat saya dimintai bantuan oleh teman saya untuk membuat Company Profile-nya. Website ini memanfaatkan Laravel sebagai frameworknya, TailwindCSS dan Blade Template sebagai styling, dan MySQL sebagai database. Sistem dalam website ini berupa monolith, dengan backend dan frontend terintegrasi menjadi satu website. 

<br />

Naru Branch Tech merupakan perusahaan start-up yang bergerak di bidang Desain dan Teknologi. Naru Branch Tech menyediakan layanan desain grafis dan pengembangan website. Website ini digunakan sebagai halaman utama Naru Branch Tech yang berisikan informasi tentang perusahaan, layanan yang disediakan, dan kontak yang bisa dihubungi.
